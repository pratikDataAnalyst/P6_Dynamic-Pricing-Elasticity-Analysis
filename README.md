# 📊 Dynamic Pricing & Elasticity Analysis

## 🎯 Objective
Analyze how price changes impact demand and revenue, and build an interactive pricing simulator.

---

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Excel
- Power BI

---

## 📈 Key Features
- Data cleaning and EDA
- Price sensitivity analysis
- Interactive dashboard
- What-if pricing simulator
